## Welcome to Database Editor Repository

This application enables you to directly edit database(`.db`) files within your web browser using a user-friendly graphical interface. You can perform operations such as adding, editing, or deleting data within your database.

First, run the development server:

### Run the development server

```bash
python server.py
```

Open (http://127.0.0.1:5000) with your browser.

## Requirements

- Python 3.x
- A `.db` (SQLite) file to edit

## Usage

1. Clone the repository.
2. Run `python server.py`.
3. Navigate to `http://127.0.0.1:5000`.
4. Upload your database file and start editing.
5. Then download edited database file.
